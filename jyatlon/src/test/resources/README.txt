In Eclipse, click on jyatlon project and select Maven > Update project...

Test scripts should have extension: .script.txt (as defined by AppTest.TEST_SCRIPT_EXTENSION).
JUnit will automatically run all the script files.

To debug a script file, change script to extension: .script.txt.tmp and execute AppTest.main()
